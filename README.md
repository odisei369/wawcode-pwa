## Instruction

Development
- git clone https://github.com/Sikora00/WawCodeCrawler
- cd WawCodeCrawler
- composer install
- php bin/console sockets:start
- configure environment
- npm install
- npm start
- goto localhost:3000

Production

- configure environment
- npm run build
- npm i serve -g
- serve -s build
- goto localhost:5000
